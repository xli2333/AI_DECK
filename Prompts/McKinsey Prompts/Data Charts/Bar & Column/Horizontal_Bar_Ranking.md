# McKinsey 横向条形图 (Horizontal Bar Chart / Ranking)

**适用场景：**
用于对项目进行排名，特别是当项目名称较长时。常见于"按类别支出"、"销售额排名"或"帕累托分析"。

**视觉风格参考：**
McKinsey6.pdf, Slide 126 / McKinsey4.pdf (General examples)

**Prompt:**
请生成一个横向条形图（Horizontal Bar Chart），用于展示排名或比较大小。

1.  **布局结构：**
    *   **方向：** 条形从左向右水平延伸。
    *   **排序：** 严格按照数值大小进行降序排列（最长的条形在顶部，最短的在底部），这有助于观众快速识别"Top drivers"。

2.  **样式与细节：**
    *   **条形外观：** 条形应显得修长、优雅，不要过粗。颜色统一使用麦肯锡蓝（McKinsey Blue）。
    *   **强调（可选）：** 如果需要强调前三名，可以将前三个条形设为深蓝色，其余设为浅灰色。
    *   **坐标轴：**
        *   左侧Y轴：列出项目名称（如Category A, Category B），文字左对齐。
        *   顶部或底部X轴：显示刻度线（0, 200, 400...），或者完全隐藏X轴，直接使用数据标签。

3.  **数据标签：**
    *   **位置：** 在每个条形的右侧末端，直接标注具体的数值。
    *   **格式：** 数值应加粗，保留合适的小数位，字体颜色与正文一致。

4.  **辅助线（可选）：**
    *   如果需要展示累计占比（如帕累托80/20法则），可以叠加一条细的折线图表示"累计百分比"，并使用右侧次坐标轴。
