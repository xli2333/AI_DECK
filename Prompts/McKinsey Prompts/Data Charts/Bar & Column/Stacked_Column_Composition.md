# McKinsey 堆叠柱状图 (100% Stacked Column Chart)

**适用场景：**
展示随时间变化或不同类别中，各部分占比构成的变化。例如：不同年份的市场份额分布，或不同产品的收入贡献占比。

**视觉风格参考：**
McKinsey General Style (隐含在市场份额分析页面)

**Prompt:**
请生成一个100%堆叠柱状图，用于展示构成的演变。

1.  **布局结构：**
    *   一系列等高的垂直柱子，代表总和为100%。
    *   每个柱子被分割成不同颜色的色块，代表不同的组成部分（如Segment A, Segment B, Segment C）。

2.  **连接线（Series Lines）：**
    *   **关键特征：** 在相邻的柱子之间，使用细灰线连接相同色块的边界。这有助于视觉引导，清晰地展示某一部分（如中间的蓝色块）是变宽了还是变窄了。

3.  **配色方案：**
    *   **单色系：** 使用同一颜色的不同深浅（如深蓝、中蓝、浅蓝、极浅蓝）。最底部的部分通常使用最深的颜色，以此类推。
    *   **对比系：** 如果要强调某一部分，将该部分设为亮色（如亮蓝），其余部分设为灰色调。

4.  **标签与图例：**
    *   **图例：** 放置在图表顶部或右侧，清晰对应颜色。
    *   **数据标签：** 在每个色块的中心显示百分比数值（如"25%"）。如果色块太小，将数值引出到柱子侧边。
    *   **顶部总数：** 尽管是100%堆叠图，有时会在柱子顶部标注当年的绝对总值（如"$50M"），以提供规模背景。
