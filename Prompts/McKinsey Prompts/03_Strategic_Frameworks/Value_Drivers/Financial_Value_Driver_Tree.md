# PPT Prompt: 财务价值驱动树 (Financial Value Driver Tree)

**类型**: 战略与框架 (Strategic Frameworks) > 价值驱动 (Value_Drivers)
**来源**: `MCK_Analysis_Part9.md` Page 238
**适用场景**: 拆解复杂的财务目标（如 ROI 或 利润），展示各驱动因素如何共同影响最终结果。

---

**Prompt 指令**:

请创建一个专业的财务驱动因素拆解图，比例为16:9。

1.  **从右至左的逻辑推导 (McKinsey Style)**:
    *   **核心结果 (最右侧)**: 放置一个最突出的数值框，展示最终指标（如 ROI 10.8x 或 Net Cash Flow）。
2.  **驱动因素层级 (向左展开)**:
    *   核心结果向左拆解为 2-3 个主要的驱动支柱（例如 "Revenue Growth" 和 "Cost Savings"）。
    *   支柱继续向左拆解为更具体的底层因子（例如 "Reduction in Absenteeism", "System Efficiency"）。
3.  **数学关系标注**:
    *   在连接线上放置数学符号（如 +、-、x、=），清晰展示各因子之间的计算逻辑。
4.  **视觉层级**:
    *   方框的大小和颜色深度应随层级变化。核心结果框最醒目，底层因子框最简洁。
5.  **色彩规范**:
    *   麦肯锡蓝系渐变。
    *   数值突出：使用加粗的深蓝色字体。
