# PPT Prompt: 标注散点图 (Annotated Scatter Plot)

**类型**: 数据图表 (Data Charts) > 散点与矩阵 (Scatter_and_Matrix)
**适用场景**: 分析两个变量之间的相关性，寻找分布模式。

**视觉风格参考**: McKinsey2.pdf, Slide 35

**Prompt 指令**:

请生成一个带有注释的散点图（Scatter Plot）。

1.  **布局结构**:
    *   **X轴与Y轴**: 清晰标注变量名称和单位（例如 Y轴="Price premium", X轴="Spend/Volume"）。
    *   **数据点**: 使用深灰色的圆形散点代表每一个样本（如SKU或客户）。

2.  **分组与注释（关键）**:
    *   **视觉分组**: 使用半透明的浅灰色椭圆或圆形背景，将聚集在一起的散点圈起来。
    *   **连接线**: 使用虚线或实线箭头，从这些分组圈引出，指向解释性文本。
    *   **解释文本**: 在图表空白处放置文本框，解释该组数据的特征。

3.  **辅助元素**:
    *   **平均线**: 添加一条虚线表示平均值或基准线（如"Average Price"）。
