# PPT Prompt: 盈亏平衡标注现金流图 (Cash Flow with Break-even Callout)

**类型**: 数据图表 (Data Charts) > 财务图表 (Financial_Charts)
**来源**: `MCK_Analysis_Part6.md` Page 211
**适用场景**: 投资回报分析，直观展示投入期、增长期及回收期。

---

**Prompt 指令**:

请创建一个专业的现金流预测图，比例为16:9。

1.  **图表主体 (垂直柱状图)**:
    *   横轴表示年份（如 2024-2030）。
    *   纵轴表示现金流数值（存在正负值）。
2.  **数据表现**:
    *   **初始负值**: 初期（如第一年）的柱子向下延伸，代表投入成本。
    *   **增长趋势**: 后续年份的柱子逐年变高且转为正值，代表收益。
3.  **关键标注 (Break-even)**:
    *   在现金流从负转正的交点位置，放置一个醒目的**黄色或橙色对话气泡 (Callout)**，内部标注 "Break-even point"。
4.  **基准线**:
    *   0 值水平线应加粗，方便识别正负。
5.  **色彩规范**:
    *   负值柱：深灰色。
    *   正值柱：深蓝色。
    *   标注：黄色。
    *   背景：纯白，带细灰色网格线。
