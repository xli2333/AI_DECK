# PPT Prompt: 敏感性分析龙卷风图 (Financial Sensitivity Tornado Chart)

**类型**: 数据图表 (Data Charts) > 财务图表 (Financial_Charts)
**来源**: `MCK_Analysis_Part5.md` Page 134
**适用场景**: 财务模型分析，展示不同假设参数变化对核心指标（如 NPV）的影响程度排序。

---

**Prompt 指令**:

请创建一个专业的敏感性分析龙卷风图，比例为16:9。

1.  **图表主体**:
    *   **中心基准线**: 页面中心放置一条垂直的 0%（或基准值）参考线。
    *   **双向条形**: 左右两侧延伸出条形，代表参数上浮/下调带来的影响。
    *   **排序 (龙卷风形状)**: 条形应按总宽度（影响程度）从上到下由大到小排列，形成漏斗/龙卷风形状。
2.  **颜色编码**:
    *   正向影响（相对于基准线向右）使用一种颜色（例如**蓝色**）。
    *   负向影响（相对于基准线向左）使用另一种颜色（例如**红色**）。
3.  **标注细节**:
    *   **左侧标签**: 列出各参数名称（如 "Churn Rate", "Price Growth"）。
    *   **数值标注**: 在条形的两端标注具体的数值或百分比。
4.  **结论区域**:
    *   在图表下方或侧边放置一个醒目的“关键风险变量”总结框。
5.  **色彩规范**:
    *   基准线：黑色或深灰实线。
    *   条形：深蓝色与深红色对比。
