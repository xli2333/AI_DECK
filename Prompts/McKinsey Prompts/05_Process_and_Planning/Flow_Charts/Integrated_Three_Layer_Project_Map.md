# PPT Prompt: 三层集成项目全景流程图 (Three-Layer Integrated Project Map)

**类型**: 流程与计划 (Process and Planning) > 流程图 (Flow_Charts)
**来源**: `MCK_Analysis_Part6.md` Page 147
**适用场景**: 咨询项目典型的全景视图，一页涵盖活动、管理沟通及交付物。

---

**Prompt 指令**:

请创建一个专业的、高度集成的项目流程图页面，比例为16:9。

1.  **三层水平结构 (从上至下)**:
    *   **顶层 (Activities, 活动流)**: 使用连续的切夫隆 (Chevron) 箭头形状连接，按阶段（如 Research, Analysis, Recommendation）划分。
    *   **中层 (Management Interaction, 管理互动)**: 放置关键节点图标，使用**菱形符号**标注重要的会议节点（如 Kick-off, Reviews）。
    *   **底层 (Deliverables, 交付物)**: 放置**圆形或矩形图标**，标注各阶段产出的具体报告或模型（如 Hypothesis, Financial Model）。
2.  **垂直对齐**:
    *   中层的会议节点和底层的交付物必须在垂直线上与其对应的上层活动阶段严格对齐。
3.  **连接与指引**:
    *   可以使用极细的垂直虚线连接三层内容，增强逻辑的一致性。
4.  **色彩规范**:
    *   活动流：麦肯锡蓝。
    *   节点图标：深灰色或橙色（用于特别突出的节点）。
