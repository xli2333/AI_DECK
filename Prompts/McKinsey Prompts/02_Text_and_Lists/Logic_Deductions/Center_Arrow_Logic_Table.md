# PPT Prompt: 中心箭头推导对照表 (Center-Arrow Logical Deduction)

**类型**: 文本与列表 (Text and Lists) > 逻辑推导 (Logic Deductions)
**来源**: `MCK_Analysis_Part4.md` Page 16
**适用场景**: 因果分析、受众分析、从现状到行动方案的策略推导。

---

**Prompt 指令**:

请创建一个专业的逻辑推导对照页，比例为16:9，强调从“发现”到“行动”的转换逻辑。

1.  **顶部双标题栏**:
    *   在页面顶部放置一个贯穿左右的深蓝色标题栏。
    *   标题栏被分为左右两部分，分别标注“现状/理解”（例如 "Understanding Audience"）和“对应策略/方案”（例如 "Tailoring Approach"）。
2.  **主体对应关系**:
    *   页面主体包含 5-6 行对应的逻辑要点。
    *   **中心连接符**: 每一行的正中心放置一个**向右指的浅蓝色实心三角形箭头**。
3.  **左右文本**:
    *   左侧文本描述现状或原因。
    *   右侧文本描述基于该现状所采取的措施。
4.  **色彩规范**:
    *   标题栏：深蓝色。
    *   连接箭头：浅蓝色或青色。
    *   文字：黑色或深灰色。
