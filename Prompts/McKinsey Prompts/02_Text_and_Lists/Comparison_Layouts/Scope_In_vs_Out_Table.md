# PPT Prompt: 项目范围界定对照表 (IS vs. IS NOT Scope Table)

**类型**: 文本与列表 (Text and Lists) > 对比布局 (Comparison Layouts)
**来源**: `MCK_Analysis_Part5.md` Page 46
**适用场景**: 项目启动会，明确工作范围，防止需求蔓延。

---

**Prompt 指令**:

请创建一个清晰的项目范围对比页，比例为16:9，采用红绿对照的视觉语言。

1.  **左右对称结构**:
    *   页面平分为左、右两部分。
2.  **内容分区**:
    *   **左侧 (In-Scope)**:
        *   标题："What this work IS"。
        *   图标：右上角放置一个**绿色的圆圈勾号 (Checkmark)**。
        *   内容：列出项目包含的任务点。
    *   **右侧 (Out-of-Scope)**:
        *   标题："What this work IS NOT"。
        *   图标：右上角放置一个**红色的圆圈叉号 (Cross)**。
        *   内容：列出明确排除在外的任务点。
3.  **视觉平衡**:
    *   左右两侧的文本框使用细灰色边框，并保持完全一致的大小。
4.  **色彩规范**:
    *   正面色：森林绿 (Green)。
    *   负面色：醒目红 (Red)。
    *   文字：黑色或深灰色。
