# PPT Prompt: “停止-开始”变革行动对照表 (Stop Doing / Start Doing Comparison)

**类型**: 文本与列表 (Text and Lists) > 对比布局 (Comparison Layouts)
**来源**: `MCK_Analysis_Part9.md` Page 234
**适用场景**: 组织转型或流程改进中，明确需要舍弃的旧习惯与需要培养的新行为。

---

**Prompt 指令**:

请创建一个直观的变革行为对照页，比例为16:9。

1.  **对比色块布局**:
    *   页面左右平分为两个大的矩形色块。
2.  **内容分区**:
    *   **左侧 (Stop Doing)**: 
        *   背景色为**深灰色或浅红色**。
        *   标题为 "STOP DOING"。
        *   列表项描述需要停止的低效行为、旧流程。
    *   **右侧 (Start Doing)**:
        *   背景色为**麦肯锡蓝或浅绿色**。
        *   标题为 "START DOING"。
        *   列表项描述新的工作方式、核心举措。
3.  **视觉引导**:
    *   可以在左右两列之间放置一个水平的向右箭头，表示行为的转变。
4.  **排版细节**:
    *   标题文字加粗反白。
    *   列表项使用简洁的无衬线体，保持左对齐。
5.  **色彩规范**:
    *   红色/灰色 vs 蓝色/绿色。
    *   文字：白色（在深色背景下）。
