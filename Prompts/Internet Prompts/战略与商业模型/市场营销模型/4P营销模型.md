# PPT Prompt: 4P 营销模型

**类型**: 战略与商业模型 > 市场营销模型
**来源**: `互联网_部分26.pdf` Page 18
**适用场景**: 制定营销组合策略：产品 (Product)、价格 (Price)、渠道 (Place)、推广 (Promotion)。

---

**Prompt 指令**:

请设计一张四分格矩阵风格的 4P 营销模型幻灯片。

1.  **中心矩阵**:
    *   在页面中央绘制一个正方形，将其均匀切割为 4 个小方块（田字格）。
    *   **左上 (Product)**: 深蓝背景，图标为盒子，文字描述产品特征。
    *   **右上 (Price)**: 深蓝背景，图标为价签，文字描述定价策略。
    *   **左下 (Place)**: 深蓝背景，图标为路径/店铺，文字描述渠道建设。
    *   **右下 (Promotion)**: 深蓝背景，图标为喇叭，文字描述推广方式。

2.  **核心圆**:
    *   在田字格的中心交汇处，叠加一个**青色的圆形**。
    *   圆内写上“用户目标”或“User Needs”，寓意所有营销策略都围绕用户展开。

3.  **外部环境**:
    *   在矩阵的最外围，画一个虚线框。
    *   在虚线框的四边分别标注外部环境因素：技术环境、社会文化环境、政治法律环境、人口经济环境。这增加了模型的深度。

4.  **底部流程**:
    *   在矩阵下方，添加一个横向的箭头流程条，总结营销的关键目标（创造价值 -> 满足需求 -> 实现盈利）。
