# PPT Prompt: 情景领导力 U 型路径

**类型**: 战略与商业模型 > 管理与流程模型
**来源**: `互联网_部分27.pdf` Page 3
**适用场景**: 领导力培训、管理风格评估。根据员工意愿和能力调整管理方式（S1-S4）。

---

**Prompt 指令**:

请设计一张直观的情景领导力模型图，重点展示 U 型路径。

1.  **坐标底图**:
    *   绘制一个标准的四象限网格。
    *   横轴：工作行为（指导行为）高 -> 低。
    *   纵轴：关系行为（支持行为）低 -> 高。

2.  **U 型路径**:
    *   这是设计的灵魂。在四象限中间，绘制一条**宽阔的、蓝色的 U 型曲线带**。
    *   路径走向：从右下象限 (S1) -> 右上象限 (S2) -> 左上象限 (S3) -> 左下象限 (S4)。
    *   在路径带上分别标注：S1 指示、S2 教练、S3 支持、S4 授权。

3.  **详细说明**:
    *   在页面的左右两侧留出空白区域。
    *   左侧：详细解释 S3 和 S4 的特征（高能力/变动意愿）。
    *   右侧：详细解释 S1 和 S2 的特征（低能力/高意愿）。
    *   使用引线将说明文字指向对应的象限。

4.  **底部补充**:
    *   在底部添加一个条形图，展示对应下属的成熟度水平（D1 - D4），与上方的 S1 - S4 形成对应关系。
