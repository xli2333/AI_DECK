# PPT Prompt: 团队组织架构图 (横向分组样式)

**类型**: 组织与人员介绍 > 组织架构图
**来源**: `互联网_部分5.pdf` Page 3
**适用场景**: 只有纯文字名单，没有照片，需要在一页内展示大量部门和人员。

---

**Prompt 指令**:

请设计一张清晰展示多部门人员名单的组织架构图。

1.  **顶部决策层**:
    *   在页面最上方，居中放置 1-2 个宽大的矩形框，代表“董事会”或“总经理”。

2.  **主体分列布局**:
    *   在下方并排设置 5-6 个垂直的**列（Column）**。
    *   每一列代表一个职能部门（如：技术部、市场部、财务部）。
    *   **列头**: 使用深蓝色背景块，写明部门名称。

3.  **人员名单堆叠**:
    *   在每个列头下方，垂直堆叠多个**浅色胶囊形小方块**。
    *   每个小方块内写一个具体的职务或姓名。
    *   方块之间保持等间距，形成整齐的视觉节奏。

4.  **视觉风格**:
    *   扁平化，利用列与列之间的留白进行分隔，无需多余的连接线。
