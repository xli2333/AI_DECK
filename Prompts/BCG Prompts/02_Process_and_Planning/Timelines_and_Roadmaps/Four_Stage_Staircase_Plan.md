# PPT Prompt: 四阶段阶梯实施计划

**类型**: 流程与时间轴 (Process and Timelines) > 路线图与甘特图 (Roadmaps and Gantt)
**文件名**: `BCG2.pdf` Page 39
**适用场景**: 用于展示从“打基础”到“全面扩张”的发展路径，强调不同阶段的工作重点转移。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9，采用阶梯状递进的时间轴布局。

1.  **顶部时间轴箭头**: 绘制一个巨大的、向右指的、分段渐变的浅绿色长箭头。标注年份区间。
2.  **阶梯阶段**:
    *   将箭头下方区域分为四个阶段，呈现出一种阶梯上升的态势（或在箭头上标注）。
    *   阶段标题：1. "Strengthen foundation", 2. "Ramp-up", 3. "Expansion", 4. "Full value capture"。
3.  **详细内容块**:
    *   在每个阶段标题下方，放置一个或多个浅绿色背景的圆角矩形块。
    *   在矩形块内，详细列出该阶段的所有关键任务（如 "Negotiate governance", "Define IT infrastructure"）。
4.  **右侧愿景链接**: 在页面最右侧（即终点处），放置一个象征“最终目标”的视觉元素（如一张代表未来的剪报或一个特殊的金牌图标）。
5.  **视觉引导**: 使用微妙的箭头或连接线，引导观众的视线从左下方的起步阶段一直移动到右上方的最终阶段。
