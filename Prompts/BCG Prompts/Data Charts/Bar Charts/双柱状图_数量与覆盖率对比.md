# PPT Prompt: 双柱状图对比 (数量与覆盖率)

**类型**: 数据图表 (Data Charts) > 柱状图 (Bar Charts)
**文件名**: `BCG1.pdf` Page 13
**适用场景**: 用于对比“绝对数量”和“相对质量/覆盖率”。例如，左边展示项目总数的增长，右边展示这些项目的质量分布。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9，布局采用左右对比的双图表形式。

1.  **页面标题**: 顶部标题栏，清晰陈述结论（例如“我们在数量增长的同时保持了高质量”）。
2.  **左侧图表 (约占页面40%)**:
    *   **类型**: 垂直柱状图。
    *   **内容**: 展示随时间（年份）累积的数量增长。
    *   **样式**: 简单的单色柱子。
    *   **标注**: 每个柱子上方直接标注具体数值。
    *   **标题**: "By Start Year" 或类似维度。
3.  **右侧图表 (约占页面50%)**:
    *   **类型**: 垂直柱状图 (或堆积图)。
    *   **内容**: 展示分布情况（如“覆盖率区间”）。
    *   **横轴**: 分组区间（如 "<65%", "65-75%", ">95%"）。
    *   **纵轴**: 项目数量。
    *   **高亮**: 使用一个红色的椭圆圈，圈出表现最好的那一组（如 ">95%" 覆盖率的柱子），并添加引出线和文本框说明（如 "22 registries achieved high coverage"）。
4.  **布局细节**:
    *   两张图表底部对齐。
    *   右侧图表可以比左侧稍宽。
    *   图表之间留有适当的呼吸空间。
5.  **配色**: 柱子使用品牌色（如灰绿色），高亮部分使用对比色（如深绿色或红色圈注）。
