# PPT Prompt: 咨询级散点图 (相关性分析)

**类型**: 数据图表 (Data Charts) > 散点图与箱线图 (Scatter and Box Charts)
**文件名**: `BCG2.pdf` Page 21
**适用场景**: 用于反驳直觉假设。例如：展示“质量”与“成本”之间并没有显著的正相关关系。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9，布局为一个大面积的散点图。

1.  **页面标题**: 顶部标题应是一个结论性的否定句（如“未发现质量与成本之间的显著相关性”）。
2.  **图表设计**:
    *   **横轴**: 性能或质量指数（如 "Quality index"）。
    *   **纵轴**: 成本数据（如 "Cost per capita"）。
    *   **散点**: 散点使用均匀的绿色圆点，每个圆点旁边附带简短的标签（如地名或项目名，Norrbotten, Skåne等）。标签位置应自动错开，避免重叠。
3.  **趋势线**:
    *   在散点群中绘制一条**水平或微降的浅灰色实线**（趋势线）。
    *   这条线不应有陡峭的坡度，以支持“无显著相关”的结论。
4.  **标注**: 在图表中央空白处直接写上一句结论性文字（如 "No significant correlation..."），文字可以加粗并带有浅色下划线。
5.  **视觉辅助**: 背景刻度线应极其淡化或省略，以保持图表的现代感和专业感。
