# Slide Layout: Detailed P&L Projection Table

**Category:** Data Insights > Financial and PnL
**Use Case:** Financial Forecasting, Budgeting, Profitability Analysis, Scenario Planning
**Source:** BCGMOREMORE_Part11 (Page 46)

## 1. Context and Goal
This slide presents a comprehensive, line-item financial projection. Unlike a simple summary table, it is designed to show the "math" behind the profit numbers, breaking down volume, revenue, costs (MACO), and final profit across multiple time periods. It emphasizes **growth trends** using visual arrows embedded within the data columns.

## 2. Visual Structure (Layout Description)
A dense, high-information table layout.
*   **Rows:** Financial line items (approx. 15-20 rows).
*   **Columns:** Time periods (e.g., "20XX H2", "20XX H1").

### Detailed Element Breakdown

#### A. The Row Headers (Left)
*   **Grouping:** Group rows into logical sections with bold headers:
    1.  **(Mn HL) Volume Metrics:** Core, Premium, Total Sales Volume.
    2.  **(Mn RMB) Revenue Metrics:** Core Revenue, Premium Revenue, Total Revenue.
    3.  **Cost Metrics:** Trade Investment, Misc Expense, MACO (Margin after Costs).
    4.  **Operational Costs:** FTE Expense, Sales Admin, IT Cost.
    5.  **Bottom Line:** Total Profit (highlighted).
*   **Indentation:** Use indentation for sub-items (e.g., "Core/Core+" under "Volume").

#### B. The Data Columns (Right)
*   **Structure:** Pairs of columns for each time period.
    *   **Value Column:** The absolute number (e.g., "47.7").
    *   **Percentage Column:** The ratio (e.g., "100%", "-29%"). Use a light grey background for these percentage columns to separate them visually.
*   **Total/Summary Rows:** Highlight key summary rows (like "Total Revenue", "Total Profit") with a light yellow or beige background spanning the full width.

#### C. The "Growth Arrows" (Visual Highlight)
*   **Location:** Placed *between* the time period columns (e.g., between "20XX H2" and "20XX H1").
*   **Style:** Small, horizontal red arrows pointing right.
*   **Content:** Above the arrow, write the absolute change (e.g., "+0.17").
*   **Function:** Visually connects the periods and highlights the incremental growth, drawing the eye to the *change* rather than just the static number.

#### D. Bottom Section (HC / Manpower)
*   **Separator:** A dotted line separating the financial data from the headcount data at the bottom.
*   **Content:** Rows for "Headcount (FTE)" to show the resource implication of the financial plan.

## 3. Key Design Instructions (For AI Generation)
*   **Density:** High. This is a "read" slide, not a "glance" slide. Text size will be small (e.g., 10-12pt).
*   **Precision:** Numbers must be right-aligned. Decimal points must align.
*   **Contrast:** Use background shading (alternating or specific for summary rows) to make the table readable.
*   **Trend Indicators:** The red arrows are the signature element. They must be perfectly aligned with the "Sales Volume" or "Revenue" rows they describe.
