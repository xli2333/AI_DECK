# PPT Prompt: 综合项目仪表板 (Dashboard)

**类型**: 结构与概览 (Structure and Overviews) > 仪表板 (Dashboards)
**文件名**: `BCG3.pdf` Page 55-59
**适用场景**: 用于作为具体案例或子项目的“名片页”，一页内展示所有核心维度信息。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9，采用极其结构化的“三段式”仪表板布局。

1.  **顶部导航**: 左上角显示项目名称和Logo，下方紧跟一句总结性副标题。
2.  **左侧视觉展示区 (约占35%)**:
    *   **布局**: 上下两层。
    *   **上层**: 放置一张高清的、代表性的实物图片或医学插图。
    *   **下层**: 放置一个较小的“关键里程碑”摘要。
3.  **右侧指标列表区 (约占60%)**:
    *   **布局**: 垂直堆叠的一系列信息条。
    *   **信息条设计**: 每一行包含左侧的一个**实色标题块**（如 "History", "Financing", "Metrics"）和右侧的详细描述文本。
    *   **视觉节奏**: 标题块颜色建议采用品牌深绿，文字白字加粗；右侧文本对齐。
4.  **底部时间轴**: 跨越页面底部的 2/3 宽度，绘制一个横向的、带箭头的年份时间轴，标注关键节点（如 2003-2009）。
5.  **配色**: 页面背景使用极浅灰以突出信息块。整体风格应呈现出“高信息密度”且“条理清晰”的特点。
