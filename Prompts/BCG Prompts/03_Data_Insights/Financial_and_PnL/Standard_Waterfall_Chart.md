# PPT Prompt: 典型瀑布图 (分析组成部分)

**类型**: 数据图表 (Data Charts) > 瀑布图 (Waterfall Charts)
**文件名**: `BCG1.pdf` Page 16
**适用场景**: 用于财务或成本分析，展示不同类别如何共同构成总额。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9，展示一个经典的财务风格瀑布图。

1.  **图表类型**: 瀑布图 (Waterfall Chart)。
2.  **构成顺序**:
    *   从左侧的第一个分类柱子开始，然后是多个连续的阶梯状悬浮柱，最后到达右侧的总计柱子。
    *   例如: "Inpatient" (起始) + "Outpatient" + "Primary care" + "Psychiatry" = "Total"。
3.  **数据标签**:
    *   每个柱子上方显示其绝对值。
    *   在悬浮柱之间的连接处上方，添加带圆圈的百分比标注（如 "25%", "41%", "~2%"），展示该项占总体的比重或覆盖率。
4.  **辅助视觉**:
    *   在图表下方添加大括号 ({)，将前几个分类括起来，并配以说明文字（如 "Specialist somatic care"）。
    *   底部留出空间放置颜色图例（如：深绿表示“已捕捉”，浅绿表示“未捕捉”）。
5.  **配色**: 瀑布图的各部分可以使用深浅不同的绿色系进行区分。
