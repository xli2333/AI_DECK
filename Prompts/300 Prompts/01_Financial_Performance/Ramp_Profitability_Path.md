# Slide Prompt: 利润率提升路径 (Profitability Ramp Chart)

**User Goal**: 
展示公司利润率（或任何核心KPI）在未来几年的提升计划，以及每个阶段的关键里程碑。

**Visual Layout**:
- **Main Visual**: 一个巨大的**上升箭头/坡道 (Rising Arrow / Ramp)**，从左下角延伸至右上角。
- **Nodes**: 在坡道上设置若干个节点（圆点），代表不同的年份（FY23, FY24... FY26）。
- **Labels**: 
  - 在每个节点上方标注当年的目标数值（如 28% -> 39% -> 45%）。
  - 在节点之间使用虚线或括弧，标注阶段性的增量（如 +5% Achieved）。
- **Supporting Text**: 在坡道的右侧或下方，列出配合该增长路径的“Progress and future initiatives”（未来举措），使用列表形式。

**Data & Content Guidance**:
- **Visual Metaphor**: 斜坡代表持续的努力和上升势头。
- **Key Data**: 必须清晰展示起点、终点和中间的里程碑。
