# 300 Series: 顶级咨询与商业汇报通用模版索引 (Master Layout Index)

本索引专为 AI 设计，旨在通过结构化的描述，帮助 AI 快速定位最适合当前商业场景的幻灯片模版。这些模版源自顶级咨询公司（如 Bain 风格）和现代科技企业的高标准 PPT 布局。

---


## 📂 01_Financial_Performance (财务与业绩)
*用于 CFO、投资者关系、经营分析会议。重点在于数字的精确性与变动原因的解释。*

| 模版文件名 (.md) | 核心用途 (Core Use Case) | 视觉特征 (Visual Layout) |
| :--- | :--- | :--- |
| **Waterfall_Net_Profit_Bridge.md** | **[归因分析]** 解释利润或成本为何发生变化（增减项分解）。 | 经典的阶梯瀑布图，绿色代表正贡献，红色代表负影响。 |
| **StepChart_Shareholder_Return.md** | **[股东回报]** 展示股息或回购金额的逐年攀升承诺。 | 带有上升箭头的柱状图或阶梯图，强调“逐年递增”的动势。 |
| **Dashboard_Financial_Summary.md** | **[经营综述]** 一页纸概览公司整体及各部门的财务状况 (P&L)。 | “表格+小图表”的高密度布局，包含损益表和关键驱动因素。 |
| **Multi_Region_Comparison.md** | **[区域业绩]** 横向对比全球各分公司/大区的经营成果。 | 多栏并列卡片，每栏包含“大数字+迷你趋势图+关键点”。 |
| **Bubble_Capital_Efficiency.md** | **[投资回报]** 评估不同业务单元的资本效率（投入 vs 产出）。 | 气泡图，X轴为资本占用，Y轴为ROE，气泡大小为利润额。 |
| **Ramp_Profitability_Path.md** | **[目标路径]** 展示利润率提升的中长期规划及里程碑。 | 巨大的上升斜坡/箭头，标注不同年份的目标值和关键举措。 |

| 模版文件名 (.md) | 核心用途 (Core Use Case) | 视觉特征 (Visual Layout) |
| :--- | :--- | :--- |
| **Map_Global_Flow.md** | **[全球布局]** 展示跨国业务流、资金流或客户迁移路径。 | 世界地图背景，使用弧线连接大洲，标注流向数据。 |
| **Donut_Market_Share_Callout.md** | **[市场份额]** 详细拆解市场构成，突出我方地位。 | 巨大的空心圆环图，通过引线连接详细的侧边说明卡片。 |
| **Columns_Growth_Trends.md** | **[多维增长]** 同时展示多个维度（如GMV、用户数、品牌力）的增长。 | 4-5列等宽布局，每列包含柱状图和大号增长百分比。 |
| **Scatter_Competitive_Positioning.md** | **[竞争格局]** 分析品牌在市场中的相对位置（如价格 vs 质量）。 | 散点图，使用虚线圈出不同的战略群组（Strategic Clusters）。 |
| **Dashboard_Regional_Opportunity.md** | **[重点区域]** 聚焦分析某一个高潜力市场（如中国/北美）。 | 地图+增长对比柱状图+优先事项列表的组合。 |
| **Demographics_Customer_Profile.md** | **[用户画像]** 详细描绘目标客群的年龄、性别、偏好等特征。 | 堆积条形图、蝴蝶图、词云的多图表组合。 |

## 📂 03_Strategic_Frameworks (战略与模型)
*用于 CEO、战略咨询项目。重点在于定性分析、逻辑推导和框架展示。*

| 模版文件名 (.md) | 核心用途 (Core Use Case) | 视觉特征 (Visual Layout) |
| :--- | :--- | :--- |
| **Matrix_Competitive_Traffic_Light.md** | **[能力对标]** 直观对比我方与竞对的优劣势（Capability Assessment）。 | 矩阵表格，使用红/绿/黄圆点或哈维球（Harvey Balls）表示强弱。 |
| **Matrix_Risk_Heatmap.md** | **[风险评估]** 评估风险的严重性与发生概率，确立优先级。 | 5x5 热力图矩阵（红/黄/绿背景），配合侧边的应对措施列表。 |
| **Model_Pyramid_Hierarchy.md** | **[层级策略]** 展示从愿景、战略到执行的自上而下结构。 | 分层的金字塔模型（3D或2D），每层右侧引出详细说明。 |
| **Model_Venn_Synergy.md** | **[交集协同]** 展示多个概念的融合或跨部门协作点。 | 三个重叠的圆形，中心交汇处高亮，强调 Synergy。 |
| **Model_Funnel_Filtering.md** | **[筛选漏斗]** 展示从海量机会中筛选出核心焦点的过程。 | 横向或纵向漏斗，展示 Input -> Filter -> Output 的过程。 |
| **Model_SWOT_Analysis.md** | **[SWOT分析]** 经典的优势、劣势、机会、威胁分析。 | 花瓣状或中心对称的四象限图形，设计感强于普通表格。 |
| **Model_Business_Canvas.md** | **[商业画布]** 系统性概括商业模式的九大要素。 | 标准的9宫格布局（价值主张、伙伴、客户、成本、收入等）。 |
