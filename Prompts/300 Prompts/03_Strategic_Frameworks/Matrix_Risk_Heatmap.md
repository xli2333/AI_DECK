# Slide Prompt: 风险热力图 (Risk Heatmap Matrix)

**User Goal**: 
评估潜在风险的严重性（Severity）和发生概率（Probability），并对关键风险进行优先级排序。

**Visual Layout**:
- **Main Visual**: 一个 **5x5 或 3x3 的矩阵 (Matrix)**。
- **Axes**:
  - X轴：Severity / Impact（严重性/影响）。
  - Y轴：Likelihood / Probability（可能性/概率）。
- **Background**:
  - 使用热力图配色：右上角（高/高）为深红色，左下角（低/低）为浅灰色或绿色，中间过渡色。
  - 背景色块应呈现平滑的弧形或阶梯状分布，而非简单的格子。
- **Data Points**:
  - 在矩阵中放置带编号的圆圈（1, 2, 3...），代表具体风险。
  - 重点风险（Top Priority）使用醒目的颜色。
- **Side Column**: 
  - 对应矩阵中的编号，列出详细的**风险缓解计划 (Risk Mitigation Plan)**。
  - 格式：Risk #1 -> Description -> Mitigation Strategy.

**Data & Content Guidance**:
- **Style**: 专业、警示性。
- **Usage**: 这是一个标准的风险管理工具。
