# Slide Prompt: 战略漏斗筛选模型 (Strategic Funnel Model)

**User Goal**: 
展示从广泛的市场机会中筛选出核心战略焦点的过程（筛选、过滤、聚焦）。

**Visual Layout**:
- **Main Visual**: 一个横向或纵向的**漏斗 (Funnel)**。
- **Structure**:
  - **入口 (Wide End)**: "Global Opportunities" 或 "Long List"。
  - **中间层 (Middle)**: "Feasibility Filter" 或 "Strategic Fit"。
  - **出口 (Narrow End)**: "Key Priorities" 或 "Action Plan"。
- **Content**:
  - 漏斗内部可以包含几个矩形块或气泡，代表正在被处理的选项。
  - 漏斗外部（通常是右侧或下方）列出最终的筛选结果（Results）。
- **Details**: 使用箭头表示流动方向。

**Data & Content Guidance**:
- **Metaphor**: 过滤杂质，提炼精华。
- **Visuals**: 可以是 3D 漏斗或 2D 扁平漏斗。
