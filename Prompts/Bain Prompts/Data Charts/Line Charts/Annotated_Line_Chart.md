# PPT Prompt: 带注释区间折线图 (Annotated Line Chart with Intervals)

**类型**: 数据图表 (Data Charts) > 折线图 (Line Charts)
**文件名**: `Bain16.pdf` Page 3
**适用场景**: 在展示长期历史数据的折线图上方，叠加不同长度的横条（区间），用于对比“历史周期”与“人员经验”或“事件持续时间”。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9。

1.  **主图表**:
    *   一个横跨页面的折线图，展示长周期的宏观数据（如 1960-2022年的 CPI）。
    *   折线颜色为灰色或深色。
    *   添加水平参考线（如平均值）。
2.  **顶部叠加层 (区间条)**:
    *   在折线图的**上方空白区域**，绘制几条不同长度的水平横条 (Bars)。
    *   这些横条代表特定的时间跨度（例如："Senior Partner Tenure", "Average Cycle Length"）。
    *   横条内或旁边注明文字，通过长度对比，直观展示当前周期相对于历史经验的特殊性（例如：横条很短，说明经验不足以覆盖当前周期）。
    *   横条使用**红色**或高亮色以示强调。
3.  **X轴**:
    *   时间轴标签稍微倾斜，以容纳密集的年份。
