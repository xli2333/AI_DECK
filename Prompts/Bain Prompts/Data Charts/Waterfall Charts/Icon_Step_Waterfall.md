# PPT Prompt: 图标阶梯瀑布图 (Icon-Topped Step Waterfall)

**类型**: 数据图表 (Data Charts) > 瀑布图 (Waterfall Charts)
**文件名**: `Bain3.pdf` Page 16, `Bain7.pdf` Page 3
**适用场景**: 用于展示市场细分或渠道的累加效应，每个阶梯代表一个具体的类别，并用图标直观表示。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9，风格生动且直观。

1.  **图表结构**:
    *   一个**阶梯状**上升的柱状图（Step Chart）。
    *   从左到右，数值逐渐累加，直到最右侧的总和。
2.  **视觉图标 (核心特征)**:
    *   在每一个阶梯柱子的**正上方**，悬浮一个圆形的图标或精美的剪切图（例如：飞机代表机场渠道，鼠标代表在线渠道，汽车代表豪车市场）。
    *   图标应足够大，清晰可辨。
3.  **颜色**:
    *   中间的阶梯柱子使用灰色或渐变色。
    *   最右侧的**总和柱子**使用醒目的**红色**，高度最高。
4.  **数据标签**:
    *   在每个柱子上方或图标旁标注具体数值。
    *   在最右侧总和柱旁，可以使用巨大的气泡显示总增长率（如 "+1%"）。
5.  **底部表格**:
    *   在X轴下方，可以附带一个小的数据表格，列出每个类别的详细增长率或CAGR。
