# PPT Prompt: 颠覆性分区矩阵 (Disruption Matrix Zones)

**类型**: 数据图表 (Data Charts) > 散点与矩阵 (Scatter and Matrix)
**文件名**: `Bain11.pdf` Page 23
**适用场景**: 用于展示不同行业受到的冲击程度，背景划分为不同的风险等级区域。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9。

1.  **背景分区**:
    *   绘制一个坐标系。
    *   **背景**: 将绘图区域水平划分为3-4个不同颜色的**带状区域**。
    *   颜色从上到下依次加深或变色（例如：顶部红色代表 "Major Disruption"，底部白色代表 "No Disruption"）。这创造了一个直观的热力图背景。
2.  **坐标轴**:
    *   X轴：互联网渗透率 (0% - 100%)。
    *   Y轴：受颠覆程度 (Disruption Index)。
3.  **数据点与路径**:
    *   使用菱形或圆形标记不同的行业（Books, Music, Luxury）。
    *   **动态路径**: 使用**红色箭头**连接两个点，展示某行业从过去（灰色点）到未来（红色点）的演变路径，表明风险等级的提升。
4.  **标签**:
    *   在左侧Y轴旁，用粗体大字标出每个背景区域的含义（如 "LIMITED DISRUPTION"）。
