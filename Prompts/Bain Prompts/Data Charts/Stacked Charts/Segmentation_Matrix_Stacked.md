# PPT Prompt: 品牌/Logo矩阵堆叠图 (Brand Matrix Stacked Chart)

**类型**: 数据图表 (Data Charts) > 堆叠图 (Stacked Charts)
**文件名**: `Bain1.pdf` Page 8, `Bain5.pdf` Page 33
**适用场景**: 用于展示不同品类下的主要品牌或市场份额结构，直接将品牌Logo作为数据点展示。

---

**Prompt 指令**:

请创建一个专业的PPT页面，比例为16:9，用于展示市场份额或品牌分布。

1.  **图表结构**:
    *   这就好像是一个变体的**百分比堆叠图**，或者是紧密排列的**矩形树状图 (Treemap)** 变体。
    *   **列 (Columns)**: 代表不同的品类（如：Watches, Jewelry, Apparel）。每列宽度可以相同，也可以根据市场规模调整宽度。
2.  **内容填充**:
    *   **Top Players**: 在每一列的底部或显著位置，使用黄色或高亮背景区域，代表前几名品牌（Top 5）的总份额。
    *   **Logos**: 关键在于**直接将品牌Logo**（如 Rolex, Cartier, Gucci）放置在对应的色块区域内，而不是仅使用文字。
    *   **其余部分**: 剩余的市场份额用灰色背景表示。
3.  **数据轴**:
    *   左侧显示百分比轴（0-100%）。
    *   顶部显示各类别的总市场价值（如 RMB Billions）。
4.  **图例**:
    *   底部添加图例说明，例如黄色框代表 "Top 5 brands"。
