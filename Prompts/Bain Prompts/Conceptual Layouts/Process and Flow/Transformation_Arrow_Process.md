# Bain Style Transformation Arrow Process

## Visual Description
A flow chart using chevron arrows or block arrows to depict a "From -> To" transformation or a linear process. It emphasizes the shift in strategy or operating model.

## Element Breakdown
1.  **Layout**:
    *   Left Column: "FROM" state (Current/Past).
    *   Right Column: "TO" state (Future).
    *   **Connector**: Large Red Arrows or Chevrons pointing left-to-right connecting the two states.
2.  **Content**:
    *   Text-heavy descriptions inside the "From" and "To" blocks.
    *   Use of bold keywords to contrast the states (e.g., "Siloed" vs "Integrated").
3.  **Icons**:
    *   Simple icons placed at the start of each row to denote the category (e.g., Technology, People, Process).
4.  **Header**:
    *   Columns are clearly labeled "TODAY" (or YESTERDAY) and "TOMORROW".

## Usage Context
*   Strategic transformation plans.
*   Operating model redesign.
*   Cultural shift descriptions.

## Prompt
```markdown
Create a 'From-To' Transformation slide in Bain & Company style.

Visual Layout:
- **Columns**: Two main text columns labeled "TODAY" (Grey header) and "TOMORROW" (Red header).
- **Rows**: [Number] rows representing different dimensions (e.g., Strategy, Organization, Technology).
- **Flow**: Connect the 'Today' text to the 'Tomorrow' text using large Red Chevron Arrows.
- **Content**:
  - **Left**: Description of the current pain points or status.
  - **Right**: Description of the future aspirational state (Bold keywords).
- **Title**: "[Title about the strategic shift, e.g., Transforming the Operating Model]".

Style Constraints:
- Use a grid alignment.
- The Red Arrows should be the primary visual element guiding the eye.
- Clean sans-serif text.
```
